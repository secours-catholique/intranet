Mode d'emploi

Pour insérer la vidéo dans un autre article de l'intranet :

  1. Copiez le « code à insérer dans l'article »
    (Exemples : <doc10949|player> ou <video10949>)
  2. Collez-le dans votre article

Pour télécharger la vidéo :

En fonction de votre navigateur, soit :
  1. Faites un clic droit sur la vidéo choisie dans le portfolio
  2. Cliquez sur « Enregistrer sous » ou « Enregistrez la cible du lien sous »
Soit :
  1. Cliquez sur la vidéo choisie dans le portfolio
  2. Enregistrer le fichier
